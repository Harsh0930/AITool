# 📄 LAYOUT.md  
## AI Backlink Intelligence & Outreach Assistant (Frontend Layout)

---

# 🧠 1. Design Philosophy

- Clean, modern, AI-first interface  
- Focus on **clarity → action → execution**  
- Reduce cognitive load (no clutter dashboards)  
- Mobile-first responsive design  
- Data + AI insights presented visually  

👉 Goal:  
**User should go from input → actionable backlinks within 2–3 minutes**

---

# 🎨 2. Design System

## 2.1 Color Theme (AI-driven)

Primary:
- Deep Blue: #0B1C3D (trust, intelligence)

Secondary:
- Electric Purple: #6C5CE7 (AI identity)

Accent:
- Neon Blue: #00D4FF (interaction highlights)

Background:
- Light Mode: #F8FAFC  
- Dark Mode: #0F172A  

Text:
- Primary: #111827  
- Secondary: #6B7280  

Success:
- Green: #22C55E  

Warning:
- Yellow: #F59E0B  

Danger:
- Red: #EF4444  

---

## 2.2 Typography

- Font Family: Poppins (Google Fonts)  
- Headings: Bold (600–700)  
- Body: Regular (400–500)  

👉 Sentence casing for body  
👉 Proper heading hierarchy (H1 → H4)

---

## 2.3 Icon System

- Font Awesome (light + duotone icons)
- Use icons for:
  - Backlinks  
  - Email  
  - Analytics  
  - AI insights  

---

# 📱 3. Responsive Layout Strategy

## Breakpoints:

- Mobile: 320px – 475px  
- Tablet: 768px – 1024px  
- Desktop: 1025px+  

---

## Mobile Design Rules (Critical)

- Single column layout  
- Sticky CTA buttons  
- Bottom navigation (optional)  
- Collapsible filters  
- Large tap targets  

👉 Prioritise:
Input → Results → Action buttons
